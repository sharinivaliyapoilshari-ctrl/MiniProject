# MiniProject
 Mini project of Task management system. 
 Task Management System (Python + SQLite3)
Objective
This mini-project demonstrates how to use Python and SQLite to create a simple Task Management System. It helps users register, log in, and manage their tasks using basic database operations.

Features

User registration and login

Add new tasks

View all tasks

Update task status (Pending, In Progress, Completed)

Delete tasks 

SQLite database for data storage

Command-line interface

Requirements

Python 3.13

SQLite3 

How to Run

Save the project file as miniproject.py.

Run the program in a terminal

Follow the CLI instructions to register, log in, and manage tasks.

Database Schema
Table: users

user_id,
username,	
password	.

Table: tasks

task_id,
user_id,
title,
description,	
status,	
due_date.	

Example Flow
Register a new user.

Log in with your username and password.

Add a few tasks with due dates.

View tasks, update their status, or delete ones you no longer need.

Log out and exit the system when finished.
